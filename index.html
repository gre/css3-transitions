Animer un site web ?

Quel intérêt ?

Les animations apportent une dynamique à une application web.
Bien utilisés, ils augmente la qualité et l'ergonomie d'une Interface Utilisateur.

  • années 2000 - Adobe Flash est le précurseur des animations un peu plus poussé.
    Il est encore beaucoup utilisé → Exemple de citroen.fr
  • 2006 - jQuery, facilite la réalisation d'animation en Javascript.
  • 2007 - Silverlight, Microsoft
  • ...
  • années 2010 - Les navigateurs sont maintenant plus puissants.
    HTML5 et CSS3, une alternative aux RIA ?

  • Les transitions (supportés sur tous les navigateurs modernes)
    → http://www.w3.org/TR/css3-transitions

Nombreuses possibilités

Implémentation

Identifier

  • Des états → sélecteur CSS
  • Des actions → exemple: :hover , :focus, ajout d'une classe...

Décrire

  • Le type de transition → transition-property
  • La durée de la transition → transition-duration
  • Le délai avant le début de transition → transition-delay
  • La variation de vitesse de la transition → transition-timing-function

Exemple

On veut changer la couleur d'une zone au passage de la souris

zone 
#zone {
  padding: 10px;
  background: red;
}
#zone:hover {
  background: blue;
}

Ajout de la propriété transition

zone 
#zone {
  padding: 10px;
  background: red;
  transition: 1s;
}
#zone:hover {
  background: blue;
}

Différents propriétés CSS

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay

transition-property

Spécifie les propriétés CSS qui doivent être animés.

transition-property: color;
transition-property: color, font-size;
transition-property: all; /* défaut */

transition-duration

Spécifie la durée de la transition.

100ms
1s
3s

transition-timing-function

Spécifie la courbe d'évolution de la transition au cours du temps.

source: http://www.w3.org/TR/css3-transitions/

transition-timing-function

ease
linear
1*
2*
  1. cubic-bezier(0.5, 2, 1, 1)
  2. cubic-bezier(0, 1, 1, 0)

transition-delay

Spécifie le délai avant le début de la transition.

100ms
1s
3s

La propriété CSS transform

permet de déplacer, zoomer et tourner un élément HTML.

Exemple

transform: translate(-100px, 50px);

transform: scale(2);

transform: scale(0.5);

transform: rotate(45deg);

transform: skew(50deg);

transform: translate(-100px, 50px) scale(0.5) rotate(30deg);

Ajout des transitions

transform: translate(-100px, 50px) scale(0.5) rotate(30deg);

transform: rotate(1000deg) scale(0);

Implémentation du diaporama d'images

Un peu de Javascript

var TIMEOUT_SLIDE = 4000;
$('.slide-images').each(function() {
  var slides = $('.slide-image', this);
  
  /// Slide Navigation ///
  var currentSlideNum = 0;
  var goToSlide = function(slideNum) {
    slides.eq(currentSlideNum = slideNum).addClass('current').siblings().removeClass('current');
  };
  var nextSlide = function() {
    goToSlide(currentSlideNum+1>=slides.size()-1 ? 0 : currentSlideNum+1);
  };
  var prevSlide = function() {
    goToSlide(currentSlideNum<1 ? slides.size()-1 : currentSlideNum-1);
  };
  goToSlide(0);
  
  /// Navigation binding ///
  $('.prevSlide').click(prevSlide);
  $('.nextSlide').click(nextSlide);
  
  /// Auto Next Slide ///
  var lastHumanNav = 0;
  $('.prevSlide, .nextSlide, .slide-pager a').click(function(){
    lastHumanNav = new Date().getTime();
  });
  setInterval(function() {
    if(new Date().getTime() - lastHumanNav > TIMEOUT_SLIDE) /* human interact is prioritary */
      nextSlide();
  }, TIMEOUT_SLIDE);
});

Et un peu de CSS...

.slide-images .slide-image,
.slide-images .slide-image img {
  transition-duration: 1.5s;
}

.slide-images.myOwnTransition .slide-image {
  opacity: 0;
}
.slide-images.myOwnTransition .slide-image.current {
  opacity: 1;
}
.slide-images.myOwnTransition .slide-image img {
  transform: scale(2);
}
.slide-images.myOwnTransition .slide-image.current img {
  transform: scale(1);
}
