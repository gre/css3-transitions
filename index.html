Nombreuses possibilités

Implémentation

Identifier

  • Des états → sélecteur CSS
  • Des actions → exemple: :hover , :focus, ajout d'une classe...

Décrire

  • Le type de transition → transition-property
  • La durée de la transition → transition-duration
  • Le délai avant le début de transition → transition-delay
  • La variation de vitesse de la transition → transition-timing-function

Exemple

On veut changer la couleur d'une zone au passage de la souris

zone 
#zone {
  padding: 10px;
  background: red;
}
#zone:hover {
  background: blue;
}

Ajout de la propriété transition

zone 
#zone {
  padding: 10px;
  background: red;
  transition: 1s;
}
#zone:hover {
  background: blue;
}

Différents propriétés CSS

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay

transition-property

Spécifie les propriétés CSS qui doivent être animés.

transition-property: color;
transition-property: color, font-size;
transition-property: all; /* défaut */

transition-duration

Spécifie la durée de la transition.

100ms
1s
3s
